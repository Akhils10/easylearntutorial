GDC 221: Introduction to Artificial Intelligence

Game Development Course 221 introduces you to basic concepts related to AI that can be added to your games. The example game we've build to illustrate the concepts taught in this course is a metaphor for Capcom's Mega Man series for Nintendo's NES system.

Each lesson has an accompanying level within the game that demonstrates the concept as a mini game (or an example of how it could be used in a real game).
